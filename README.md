# 🧪 KeywordLab Studio

![Header Placeholder](https://images.unsplash.com/photo-1618005182384-a83a8bd57fbe?q=80&w=2564&auto=format&fit=crop)

**KeywordLab Studio**, Pinterest moodboard'larınız için mükemmel anahtar kelimeleri bulmanıza yardımcı olan, yapay zeka destekli, ultra-minimalist bir anahtar kelime oluşturucudur. 

---

## ✨ Özellikler

- 💎 **Premium UI:** Asimetrik stüdyo düzeni, cam tasarımı (glassmorphism) ve "noise" dokusuyla üst düzey bir deneyim.
- ⚡ **AI Gücü:** Google Gemini 2.0/3.0 modelleri ile hedef odaklı anahtar kelime üretimi.
- 🎨 **Minimalist Estetik:** Göz yormayan, soft mor tonlarında modern bir tasarım dili.
- 📱 **Tam Responsive:** Mobil ve masaüstü cihazlar için optimize edilmiş arayüz.
- 📋 **Hızlı Kopyalama:** Üretilen kelimeleri tek tıkla panoya kopyalama.

## 🚀 Teknolojiler

- [Next.js](https://nextjs.org/) (App Router)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Google Gemini AI API](https://ai.google.dev/)
- [Shadcn UI](https://ui.shadcn.com/)
- [Lucide Icons](https://lucide.dev/)

## 🛠️ Kurulum

1. Depoyu klonlayın:
```bash
git clone https://github.com/rasperon/anahtar-kelime-olusturucu.git
```

2. Bağımlılıkları yükleyin:
```bash
bun install
# veya
npm install
```

3. Geliştirme sunucusunu başlatın:
```bash
bun dev
```

4. Uygulama içindeki **Ayarlar** panelinden Gemini API anahtarınızı girin.

## 🔑 API Anahtarı Nasıl Alınır?

[Google AI Studio](https://aistudio.google.com/) üzerinden ücretsiz bir API anahtarı alarak hemen kullanmaya başlayabilirsiniz.

# English Description

**KeywordLab Studio** is an AI-powered, ultra-minimalist keyword generator designed to help you find the perfect search terms for your Pinterest moodboards.

### Features
- **Premium UI:** Asymmetric studio layout with glassmorphism and grain textures.
- **AI Powered:** Leverages Google's Gemini 3 Flash models for targeted keyword generation.
- **Quick Export:** One-click copy to clipboard functionality.

### Getting Started
Simply clone the repo, run `bun install` & `bun dev`, and enter your Gemini API key in the in-app settings modal.
